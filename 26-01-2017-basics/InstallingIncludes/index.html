Installing Includes and Plugins
1: Installing includes In this guide we will be installing the AeroLib include and using firefox. To start scroll down the post till you find the downloads section. Download the AeroLibrary include only zip. Once Downloaded Follow these steps: 1:Click the arrow down button at the top of firefox, then click the little folder icon next to the file you just downloaded. That will open up your downloads folder. 2: Right click the folder and click cut. Then go to your simba/includes folder. 
C:\Simba\Includes
1:Then Click in a white spot inside that folder, and click paste. 2:After that ,right click the folder and extract it, then rename that folder to just AeroLib 3: The zip is no longer need, so you can delete it. 2: Installing Plugins To install the plugins: Download the plugins only link on the thread. 1:Then ,follow the same steps as before, but this time paste the file in to your plugins folder 
 C:\Simba\Plugins
2: After you have pasted the folder, in the correct location, extract it and delete the zip.
